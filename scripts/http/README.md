# API request collections

Executable HTTP request collections for hitting the API by hand during
development. These are tooling, not documentation.

- `area-iq-api-tests.http` — VS Code REST Client format (primary).
- `area-iq-api-tests-httpyac.http` — httpyac variant of the same set
  (redundant; kept for httpyac users — pick one format and drop the
  other when convenient).

Set `@baseUrl` / auth token at the top of each file before running.
